## About The Project

This project provides libraries for the PS25454 capacitive sensor. The library includes the schematic symbol, PCB footprint, and 3D model for the PS25454 component. Use this library in your Eagle projects to add the PS25454 to your designs.

<!-- GETTING STARTED -->
## Getting Started

To get started with using the PS25454 Eagle library, follow the instructions below:

### Prerequisites

* [Autodesk Eagle](https://www.autodesk.com/products/eagle/overview)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/infocbra/PS25454-socket.git
   ```
2. Copy the library files to your Eagle libraries directory.

3. Add the library to your Eagle project by going to `Library > Open library manager > Available` and browsing to the location where you copied the library files.

<p align="right">(<a href="#readme-top">back to top</a>)</p>

<!-- USAGE -->
## Usage

This Eagle library includes the schematic symbol, PCB footprint, and 3D model for the PS25454 component. Use this library in your Eagle projects to add the PS25454 to your designs.

### How to Add the Component

1. Open your Eagle project.
2. Click on `Add Part` and find `PS25454` in the list of available components.
3. Place the component in your schematic.
4. Connect the component pins to your circuit as needed.
